=============================
遥感式路面传感器用户指南
=============================

-----------------------------
1、产品介绍
-----------------------------
遥感式路面传感器，采用了遥感技术，避免了对道路的破坏，从而不会因为安装道路气象站而引起对交通的干扰。利用多光谱测量技术能够准确检测出道路表面状态，比如结冰、积雪和积水，以及对应的厚度。
在埋入式路面传感器不便或不能安装的路面条件下，遥感式路面传感器是最理想的选择。遥感安装，意味着不需要封闭道路、不需要切割路面，安装工作既安全又方便。维护量少，是道路气象系统组成中一项理想选择。它既可以安装在现有的气象站上，也可以安装在路面视野无遮挡的其他建筑物上。
遥感式路面传感器被安装在一个全天候、耐久的外壳中，以保证承受恶劣天气，这使得它在任何天气条件下能提供精准数据。
通过提供路面条件信息，遥感式路面传感器为道路管理部门提供准确的监测数据，在道路安全出现险情之前，采取相应措施。
 
 - 功能
 
  - 检测路面积水、积雪、结冰厚度
  - 远距离遥感检测路面状况
  - 非埋入式安装快捷简便
  - 维护成本低
  - 融入现有公路自动气象监测网络
 
 - 应用
 
  - 远距离安装
  - 桥面
  - 事故多发区域
  - 车流量大的区域
  - 雨雪多发地区
 
 - 特点
 
  - 远距离测量路面积水、冰和雪
  - 测量路面状态
  - 测量积水、冰面、湿滑程度
  - 非埋入式设计
  - 抗锈蚀
  - 红外检测最远 10 米
  - 无需封闭车道，安装维护简单
  - 坚固设计，全天候测量

2、主要指标
-----------------------------

=================  =================================================
监测距离           2-10米
检测直径           30 cm
角度               10-70度
电源输入           9-30VDC
最大功耗           4W（非除雾加热状态）
工作温度           -40 ºC 至 +70 ºC
工作湿度           0 至 100%
路面状态输出       水：0.00—10mm
\                  冰：0.00—5mm
\                  雪：0.00—10mm
\                  湿滑程度：0.01—1
精度               0.01mm
镜头污染检测       光学镜头的污染等级测量及内部自动污染补偿
路面               混凝土、沥青路面
通讯               RS485
平均无故障         1.5 x 1000000 小时
安全               无安全问题 – 红外遥感测量技术

-----------------------------
3、对外接口
-----------------------------
 传感器对外部提供6根通信线，定义分别为：

        ========  ========  ========  ========  ========  ========  ========
        颜色      红        黑        黄        白        蓝        绿
        定义      电源正    电源负    RS485A    RS485B    按键      按键
        ========  ========  ========  ========  ========  ========  ========

^^^^^^^^^^^^^^^^^^
通讯端口
^^^^^^^^^^^^^^^^^^
 物理端口定义

        - 起始位——1bit
        - 停止位——1bit
        - 数据位——8bits
        - 校验位——none
        - 波特率——9600bps
	
^^^^^^^^^^^^^^^^^^
端口连接
^^^^^^^^^^^^^^^^^^		
 仪器与外部设备之间通过RS485 进行。

^^^^^^^^^^^^^^^^^^
通信协议
^^^^^^^^^^^^^^^^^^
 传感器支持ASCII协议以及Modbus RTU协议，具体通信格式参考对应文档。

-----------------------------
4、按键与红色激光指示灯
-----------------------------
^^^^^^^^^^^^^^^^^^
位置指示
^^^^^^^^^^^^^^^^^^
- 开启指示灯
 在红色激光指示灯关闭状态下，长按按键超过2秒再抬起，开启指示灯。该功能主要用于安装时指示测试位置点。

- 关闭指示灯
 在红色激光指示灯开启状态下，长按按键2秒再抬起，关闭指示灯。

^^^^^^^^^^^^^^^^^^
干标定
^^^^^^^^^^^^^^^^^^
 在对准要的测试的路面点后，连续按按键2次（间隔小于0.5秒），执行干标定。安装时须执行该操作，使测试数据更准确。

 标定完成时，红色激光再次点亮2秒，提示标定完成。

^^^^^^^^^^^^^^^^^^
镜头清洁标定
^^^^^^^^^^^^^^^^^^
 在镜头清洁，没有结雾和污染情况下，连续按按键6次（间隔小于0.5秒），采集镜头清洁时值，该值在镜头检测功能中作为参考。在初次安装时须执行该操作。

 操作完成时，红色激光再次点亮2秒，提示操作完成。

-----------------------------
5、安装方法
-----------------------------
- 安装位置

 安装路面传感器推荐使用直径8cm以上立柱，高度大于3m。传感器与立杆角度10 ~ 70度 。直射距离2m~10m。确保直射光路无遮挡，红色激光指示位置为沥青路面，无其他遮挡物。

- 安装步骤

	- 1. 选择天气晴朗，空气湿度低的天气进行安装
	- 2. 安装时，确保被测路面状态为干燥状态
	- 3. 按照上述推荐安装位置进行传感器固定。可以利用位置指示功能，对焦被测路面。
	- 4. 确定被测路面后，进行“干标定”，并等待标定完成
	- 5. “干标定”完成后，进行“镜头清洁标定”，并等待完成。
	- 6. 重新上电
